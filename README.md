# URYLEO Optimizer

Repositório oficial do site institucional e dos materiais públicos do projeto **URYLEO Optimizer**.

O URYLEO Optimizer reúne **ferramentas, aplicativos e soluções** focadas em:

- Otimização de sistemas  
- Produtividade no dia a dia  
- Organização e utilidades para usuários comuns e avançados  

Sempre com **linguagem transparente**, sem promessas exageradas e com foco em **utilidade real**.

## Site institucional

Este repositório contém o código do site institucional da URYLEO Optimizer.

- Arquivo principal: `index.html`
- Versão online (Netlify):  
  https://uryleo-optimizer-otimiza-o-produtivid.netlify.app
- Tecnologias utilizadas:
  - HTML5
  - CSS3
  - JavaScript (apenas scripts simples dentro do HTML)
- Nenhum framework externo (sem Bootstrap, Tailwind, etc.)
- Tema: **escuro**, visual **moderno** e responsivo (desktop e mobile)
